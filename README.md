# Vector Vault

A fast-paced, momentum-based vault run game. Navigate through challenging levels, collect data shards, and dodge rotating laser turrets to reach the exit.

## 🎮 Gameplay

Master the physics-based controls to guide your character through each level:

- **Collect Shards** — Pick up all data shards scattered throughout each level
- **Dodge Turrets** — Evade rotating laser fire from automated defense systems
- **Reach the Exit** — Safely dock at the exit point to complete the level
- **Master Momentum** — Use thrust, braking, and drifting to navigate tight spaces

## 🕹️ Controls

| Input | Action |
|-------|--------|
| **WASD** / **Arrow Keys** | Thrust directional movement |
| **Space** / **X** | Brake to slow down |
| **Shift** | Drift for precise control |
| **R** | Restart current level |
| **P** | Pause the game |
| **N** | Next level (when completed) |

## 📊 Levels

Vector Vault features multiple challenging levels with increasing difficulty:

1. **Level 1 — Training Bay** — Learn the basics
2. **Level 2 — Offset Corridors** — Navigate narrow passages
3. **Level 3 — Pinball Atrium** — Dodge dynamic obstacles
4. **Level 4 — Vault Cycle** — Master advanced techniques

Each level tracks your best time, so compete with yourself to improve!

## 🚀 Getting Started

1. Open `index.html` in your web browser
2. Follow the on-screen instructions
3. Use the controls to complete each level
4. Try to beat your previous best times!

## 💾 Progress

Your completion times are automatically saved in your browser's local storage, allowing you to track your best performances across sessions.
